Screenplay is a cloud video editor with a unique text interface. Users can upload videos, receive an automatic transcription, and edit the video by interacting with the text transcription like a text editor.

  • Express backend running on Google Cloud App Engine
  • Complex Angular frontend with dynamic user interface built from scratch
  • Video conversions (ffmpeg) and video rendering (MLT Framework) performed with microservices / serverless cloud functions (Lambda)
  • Connects to IBM Watson's speech recognition API to generate video transcriptions
  • Custom responsive design using Sass
  • My final project at Ironhack coding bootcamp, winner of the Ironhack August 2018 hackshow.

Pronuncio is a small game made after 2 weeks of bootcamp training in HTML, CSS & JS. The aim is to pronounce sentences in different languages, with your attempts transcribed via speech to text, and scored based on accuracy.

  • Connects to Google Translation API
