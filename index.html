Screenplay is a cloud video editor with a unique text interface. Users can upload videos, receive an automatic transcription, and edit the video by interacting with the text transcription like a text editor.

My final project at Ironhack coding bootcamp, and the winner of the final hackshow.

  • Express backend running on Google Cloud App Engine
  • Complex Angular frontend with dynamic user interface built from scratch
  • Video conversions (ffmpeg) and video rendering (MLT Framework) performed with "serverless" cloud functions
  • Connects to IBM Watson's speech recognition API to generate video transcriptions
  • Custom responsive design using Sass

PackApp is an open source, cloud based build service that lets you package your web app or website into a WebView based Android APK.

  • Angular frontend
  • Express + Mongo backend
  • Linux build server building java Android apps with gradle and shell scripts

An open source netflix clone built in a pair at Ironhack coding bootcamp.

  • Server rendered Express app

A small game made after 2 weeks of bootcamp training in HTML, CSS & JS. The aim is to pronounce sentences in different languages, with your attempts transcribed via speech to text, and scored based on accuracy.
