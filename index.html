My final project at Ironhack coding bootcamp, the winner of the final hackshow. Built with a MEAN stack, VidPad is a web app which aims to make video editing in the cloud as easy to use as a notepad.

  • Video uploads to Google Cloud Compute
  • Conversions on server using ffmpeg
  • Videos are transcribed with IBM Watson's speech recognition API
  • Users can edit videos by highlighting and reordering the transcribed text
  • Video rendering performed on server using Melt Framework
  • Asynchronous functions and node child processes
