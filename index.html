My final project at Ironhack coding bootcamp, and the winner of the final hackshow. Built with the MEAN stack, VidPad is a web app which aims to make video editing in the cloud as easy to use as a notepad.

  • Backend running on Google Cloud Compute
  • Video uploads
  • Video conversions on server using ffmpeg
  • Videos are transcribed with IBM Watson's speech recognition API
  • Users can edit videos by highlighting and reordering the transcribed text
  • Video rendering performed on server using Melt Framework
  • Uses many asynchronous functions and node child processes
  • Custom responsive design using Sass
